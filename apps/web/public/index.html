Comments that nobody can delete.

A browser extension and embed protocol that layers decentralized comments on top of any page on the internet. One human, one account, anchored to a biometric — no platform bans, mute and block live on your device.

Base L2 WebAuthn Semaphore ZK IPFS / Arweave MIT

Why bother

No platform bans

The protocol has no admin role and no delete primitive. Removing a comment is not a button anyone can press.

Bot-free reviews

Every account is a verified human via WebAuthn + Semaphore proof of group membership.

Mute is yours

Block lists are client-side. Optionally publish them on-chain so they travel between your devices and clients.

Permanent

Comments live on IPFS, anchored on Base, archived to Arweave. There is no host that can vanish.

SDK demo

Drop-in embed any site can ship. Zero runtime dependencies, ~5KB minified. The fixture below points at a non-existent indexer endpoint, so you'll see the "could not load" path — that's the unhappy state working as designed. 

<div id="fos-comments"></div>
<script src="/sdk.js" data-target="#fos-comments" data-theme="auto"></script>

Run it locally

git clone https://github.com/Freedom-Guda/freedom-of-speech.git
cd freedom-of-speech
pnpm install
pnpm --filter @fos/contracts demo:inline   # full E2E in Hardhat memory
pnpm --filter @fos/extension build         # then load .output/chrome-mv3 in Chrome
pnpm --filter @fos/web dev                 # this page, hot-reload
